### Was ist ein mount point? In welchen beiden Systemkommandos wird dieser benötigt? Erläutern Sie kurz die Aufgaben dieser beiden Kommandos.

Ein Mountpoint ist ein Verzeichnis in einem Unix-Betriebssystem, in welches ein angeschlossener Datenträger eingehangen, d.h. dem System zugänglich gemacht wird.

Die beiden zusammenhängenden Kommandos sind `mount` und `umount`.

Mittels

`mount /dev/sdb1 /mnt` wird der Datenträger, der vom System sdb genannt wurde, mit der ersten Partition am Mountpoint /mnt eingehangen.

Dadurch kann das System über den Pfad /mnt auf die Partition sdb1 zugreifen.

Mittels `umount /mnt` wird der Datenträger am Mountpoint /mnt wieder ausgehangen, d.h. das Syste trennt die Datenverbindungen zum Datenträger, so dass er ohne Datenverlust physikalisch getrennt werden kann.
